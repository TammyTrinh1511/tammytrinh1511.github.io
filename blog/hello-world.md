This is a template post. Copy this file to start a new one, then add a matching
entry to `blog/posts.json`. The post **title and date come from the manifest**, so
start straight with your content here; no need to repeat the title.

## Writing in Markdown

You can use everything standard Markdown offers:

- **bold**, *italic*, and `inline code`
- [links](https://tammy-trinh.github.io)
- lists, like this one

> Blockquotes work too, handy for highlighting a key idea.

### Code blocks

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("world"))
```

### Images

Drop image files in `blog/assets/` and reference them relatively, like
`![caption](assets/my-image.png)`.

That's it: write, save, add a manifest entry, and the post appears on the home page.
